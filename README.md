================================================================================
Polygonology
--------------------------------------------------------------------------------
Written in Love2D (0.10.0) for Ludum Dare 35

--------------------------------------------------------------------------------
Collect the greens, 
Dodge the reds, 
as many times as possible 
before time runs out.

--------------------------------------------------------------------------------
Controls
* w				- Up
* a				- Left
* s				- Down
* d				- Right
* space			- Deflect (todo)
* left shift	- Boost   (todo)
* esc			- Quit
* return		- Select Menu Item
* f1			- Save Score 
* f11			- Toggle Fullscreen
* f12			- Grab Screenshot
